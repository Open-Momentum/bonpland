# Bonpland

## Sistema de Gestión de Propiedades

### Inmobiliaria Bonpland Propiedades

**Contexto:** Muchas inmobiliarias todavía gestionan sus propiedades de forma manual. Este proyecto busca digitalizar su oferta para mejorar la visibilidad y el contacto con potenciales clientes.

**Objetivo:** Permitir que los usuarios busquen inmuebles aplicando filtros avanzados y que la inmobiliaria gestione las publicaciones desde un panel privado.

### Requerimientos (MVP)

- Página de inicio con propiedades destacadas.
- Sección “Quiénes somos” y “Trayectoria”.
- Página de contacto con datos de la inmobiliaria.
- Listado de inmuebles con filtros: ambientes, país, ciudad, metros cuadrados, rango de precios y tipo de contrato (alquiler o compra).
- Página de detalle de inmueble con opción para contactar al negocio usando el código de referencia.
- Panel privado para gestión de inmuebles (crear, editar, eliminar).
- Estados posibles: Disponible, Reservado, Alquilado, Vendido.

#### Definición de completado

El proyecto estará completo cuando los usuarios puedan navegar, filtrar y contactar por un inmueble, y los administradores puedan gestionar el inventario desde el panel.
